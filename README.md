# 5568-front-end

## Recommended Browser

- Chromium-based browsers (Chrome, Edge, Brave, etc.)
- Firefox

## Project Setup

1. Install **Node.js**

   <a>https://nodejs.org/en/download</a>

2. Install Requirements(In the terminal)

   ```sh
   npm install
   ```

3. Compile and Hot-Reload for Development

   ```sh
   npm run dev
   ```

4. Open the link in showed in the terminal
